Platform Game (Unfinished Game Development)\n

  * Move ←→           —   A D
  * Jump              —   Space
  * Atack             —   LMB (Left Mouse Button)
  * Raise the shield  —   RMB (Right Mouse Button)
  * Dash              —   Ctrl
